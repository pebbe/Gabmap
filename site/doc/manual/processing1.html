Type of processing

You can upload four different types of data in Gabmap:

  • string data (phonetic transcriptions)
  • numeric data (e.g., formant values)
  • categorical data (e.g., choice of allomorph)
  • difference data (matrix with pair-wise linguistic distances between the sites)

For the different types of data, you can choose between different processing types. The type of processing determines how the linguistic distances between the places in the data set are computed. The different processing types are described in the following sections:

  • String edit distance (=processing of string data)
  • Processing of numeric data
  • Processing of categorical data

No processing type is needed for difference data, because this input as such comprises distances between places.
