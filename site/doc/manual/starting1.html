Getting started

To start using Gabmap, you first you have to create an account by choosing a username and a password. A message will be sent to you by email when you have created your account. Follow the instructions in the email to confirm the account. If you do not use an account for more than 2 months it will be deleted automatically.

Once you have logged in, you can create a new project in Gabnap by:

  1. Giving it a describing name
  2. Uploading a map file
  3. Uploading a data file
  4. Choosing data type and type of processing
  5. Clicking 'Create project'
