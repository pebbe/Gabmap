A regular expression (RE) can contain both special and ordinary characters.
Most ordinary characters, like A, a, or 0, are the simplest regular expressions; they simply match themselves. You can concatenate ordinary characters, so last matches the string 'last'.

The special characters are:

. Matches any character.
Example: a.z matches 'abz' and 'acz', but not 'az' or 'abcz'.
^ Matches the start of the string.
Example: ^a matches 'ab', but not 'ba'.
$ Matches the end of the string.
Example: z$ matches 'yz', but not 'zy'.
* Matches 0 or more repetitions of the preceding RE.
Example: ab*c matches 'ac' and 'abc' and 'abbc' and 'abbbc'.
Example: a.*z matches 'az' and 'abcz' and 'abcdefz'.
+ Matches 1 or more repetitions of the preceding RE.
Example: ab+c matches 'abc' and 'abbc' and 'abbbc', but not 'ac'.
? Matches 0 or 1 of the preceding RE.
Example: ab?c matches 'ac' and 'abc', but not 'abbc' or 'abbbc'.
{{m,n}} Matches from m to n repetitions of the preceding RE.
Example: ab{{2,4}}c matches 'abbc' and 'abbbc' and 'abbbbc', but not 'abc' or 'abbbbbc'.
\ Escapes special characters.
Example: \? matches '?'.
[] Indicates a set of characters.
Example: [a-z124680] matches any lower case letter or a single digit even number.
A ^ as the first character indicates a complementing set.
Example: [^0-9] matches anything that is not a digit.
Example: ^[^0-9]*$ matches any string that does not contain a digit.
| A|B, creates an RE that will match either 'A' or 'B'.
(...) Matches the RE inside the parentheses.
Example without parentheses: abc*d matches 'abd' and 'abcd' and 'abccd' and 'abcccd'.
Example with parentheses: a(bc)*d matches 'ad' and 'abcd' and 'abcbcd' and 'abcbcbcd'.
Example without parentheses: ab|cd matches 'ab' and 'cd'.
Example with parentheses: a(b|c)d matches 'abd' and 'acd'.
