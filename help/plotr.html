Example usage in R: 

    Sys.setlocale(locale="en_US.UTF-8")    ## for non us-ascii in project description
    load("gabmap.Rdata")                   ## load the data
    ls(pattern="gabmap")                   ## see what is in it
    plot(gabmap.Data, main=gabmap.Description, pch=".")
    lines(gabmap.Mean, col="red", lwd=2)
