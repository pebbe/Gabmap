Value maps

Value maps are created for numeric data only. The maps are like a diagram of the input data in map form, and help you to inspect the geographic distribution of values on each variable.

Choose a varaible in the drop down menu and click Select item. The map displays the value at each place of the chosen variable. In the map the numeric data in the input table are converted to colors: the darker the color on the map the higher the value (see whole color range).
