Map file

You can create your map using Google Earth. With Google Earth you can draw the borders of your area (add one ore more polygons) and add placemarks for the locations where the data was collected. Note that the place names have to be spelled exactly as in your data file. Optionally, you can add more details, like internal borders or rivers, to the map. When you have made the map in Google Earth, you save it as a .kml file or .kmz file, which can be uploaded directly in Gabmap.

A short manual for making a map with Google Earth is available here. If you follow this manual you can skip the last step (Convert to RuG/L04), because the manual was made for the older Groningen software RuG/L04, but in Gabmap you can upload the .kml file directly.

If you already have a list of locations with coordinates, you can find a tool here for converting it to .kml. Then you can open the converted file in Google Earth to add borders.

When a project has been created in Gabmap, an index map is created, and the map can be inspected and downloaded as an image file.

On several of the maps created in Gabmap each data point is represented by a colored polygon. The area of this polygon is computed automatically in Gabmap using Delaunay triangulation.
